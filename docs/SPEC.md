## 一、日常工具

### 1.1 计算器 ✅

**核心能力：**
- 基础四则运算：+ − × ÷
- 百分比计算
- 科学计算：sin/cos/tan（角度制）、平方根、log₁₀/ln、平方/立方、倒数、阶乘、绝对值、幂运算
- 退格、清空，科学模式可折叠

**边界情况：**
- 除以 0 → 提示"除数不能为 0"
- 负数开平方 / 0 的倒数 → 提示
- 阶乘 > 170 → 提示"数值过大"

---

### 1.2 单位换算 ✅

**核心能力：**
- 8 大类：长度、重量、温度、面积、体积、数据、速度、时间
- 60+ 单位，温度特殊公式换算
- 双向换算，一键切换源/目标

**分类详情：** 长度(mm~ly) / 重量(mg~st) / 温度(°C/°F/K) / 面积(mm²~ac) / 体积(mL~ft³) / 数据(B~TiB) / 速度(m/s~Mach) / 时间(ms~yr)

**边界：** 空输入→空结果，浮点数保留 6 位有效数字

---

### 1.3 日期计算 ✅

**核心能力：** 日期差（含工作日）、加减年/月/日（月末截断）、年龄计算

**边界：** 闰年正确处理，1月31日+1月→2月28/29日

---

### 1.4 倒计时 ✅

**核心能力：** 设定目标日期，实时秒级刷新，中文描述

**UI：** 天 72px + 时:分:秒 48px，已过变橙色

---

### 1.5 随机选择器 ✅

**核心能力：** 列表随机选 / 抽签（可指定数量）/ 随机数字 / 抛硬币 / 随机颜色

---

### 1.6 字数统计 ✅

**核心能力：** 字符/单词/行数/中文字数/字节/数字/字母/标点/空格 实时统计

---

### 1.7 番茄钟 ✅

**核心能力：** 25min 工作 / 5min 休息循环，4个番茄后长休息。自定义时长。圆形进度条 + 阶段计数

---

### 1.8 秒表 ✅

**核心能力：** 计时/暂停/重置，计圈功能（圈数记录），毫秒精度

---

### 1.9 房贷计算器 ✅

**核心能力：** 等额本息 / 等额本金。输入总额、年利率、年限。输出月供、总利息、总额

---

### 1.10 节假日 ✅

**核心能力：** 中国法定假日（春节/清明/端午/中秋等），最近节日倒计时，年份切换浏览。支持 2024-2030 农历数据

---

### 1.11 纪念日 ✅

**核心能力：** 自定义多个纪念日，支持倒数日/已过天数两种模式，自动周年计算，秒级刷新

---

### 1.12 世界时区 ✅

**核心能力：** 17 个全球时区实时时间，搜索过滤（城市/时区名/代码），与北京时间时差，标注本地时区

---

### 1.13 Emoji 搜索 ✅

**核心能力：** 200+ Emoji 按 7 个分类浏览，关键词搜索，点击复制到剪贴板

---

### 1.14 投资计算 ✅

**核心能力：** 复利终值 / 每月定投 / 总投入 vs 总收益。输入本金、月投、年化率、年限

---

### 1.15 二维码生成 ✅ 

**核心能力：** 文本/链接 → QR 码（CustomPainter 绘制），纠错级别 H（30%），实时输入即生成

---

## 二、格式化 & 转换

**核心能力：** 格式化（2/4空格）/ 压缩 / 校验 / 示例填入

**核心能力：** 文本↔Base64，URL Safe 模式

**核心能力：** encodeURIComponent / encodeURI，混合编码智能识别

### 2.4 时间戳工具 ✅

**核心能力：** 秒/毫秒↔日期，ISO 8601 / 中国 / 美国格式，相对时间。实时双向转换

### 2.5 颜色工具 ✅

**核心能力：** HEX / RGB / HSL / HSV 互转，颜色名称匹配，简写 HEX 展开，色块预览

---

### 2.7 XML 格式化 ✅

**核心能力：** 标签级缩进 / 压缩。处理自闭合标签、XML 声明

---

### 2.8 YAML ↔ JSON ✅ 

**核心能力：** 双向转换，自动检测输入格式

### 2.9 配色方案 ✅

**核心能力：** 15 种预设调色板（Material/日系/莫兰迪/赛博朋克等），互补色 / 类似色 / 三分色生成，自定义颜色输入。点击复制 HEX

## 三、编码 & 加密

### 3.1 UUID 生成器 ✅

**核心能力：** UUID v4 / v7，NanoID，批量（1-100），大写/小写/去横线/花括号

---

### 3.2 哈希工具 ✅

**核心能力：** MD5 / SHA-1 / SHA-224 / SHA-256 / SHA-384 / SHA-512，实时多算法输出，大写切换

---

### 3.3 JWT 解码器 ✅

**核心能力：** 解析 Header/Payload/Signature，格式化 JSON，过期检测（红色/绿色状态条）

**边界：** 非三段式→提示，过期→红色警告

---

### 3.4 进制转换 ✅

**核心能力：** 2 / 8 / 10 / 16 进制实时互转，验证输入合法性，多结果同时展示

---

### 3.5 密码生成器 ✅

**核心能力：** 可配置长度（4-64）、字符集（大小写/数字/符号）、易混淆排除。强度评分（弱/中等/强/非常强），一键复制

---

## 五、收藏系统（已完成）

### API

| 方法 | 路径 | 功能 |
|---|---|---|
| GET/POST/DELETE | `/api/user/favorites` | 收藏 CRUD |
| PUT | `/api/user/favorites/reorder` | 排序 |
| PUT | `/api/user/favorites/:toolId/move` | 移动文件夹 |
| GET/POST | `/api/user/favorites/folders` | 文件夹列表/创建 |
| PUT/DELETE | `/api/user/favorites/folders/:name` | 重命名/删除 |

### 前端特性
- 收藏时文件夹选择弹窗（有文件夹则弹出，没有则直接收藏）
- 文件夹管理：新建/重命名/删除
- 收藏项移出到其他文件夹

## 六、UI/UX 规格

### 布局
- **宽屏 ≥640px**：NavigationRail + 分类列表 + 工具内容（打开工具后只显示内容）
- **窄屏 <640px**：底部 NavigationBar + AppBar + 抽屉切换工具

### 暗色模式
- 跟随系统，所有工具支持深色/浅色

### 工具打开
- 从 Dashboard 点工具 → 全屏工具内容（仅有返回按钮，无侧栏）
