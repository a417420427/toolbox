# 📋 SPEC.md — 个人效率工具箱 功能规格

> 本文档定义每个工具的具体行为、输入输出、边界情况、UI 要求和端侧分布。
> 是写代码时对照的"最终规格"，不做模糊设计。
>
> ✅ = 已完成 · 📌 = 计划中

---

## 符号说明

- **✅ 纯前端** — 完全在浏览器/Flutter 本地执行，不需要网络
- **🔷 需后端** — 需要服务端资源（大文件/库不支持前端/Python 生态）
- **🧩 可离线** — PWA/Flutter 本地缓存后可用
- **📌 V2+** — 非 MVP，后续版本加入

## 零、日常工具类（新增 — MVP 实装）

### 0.1 计算器 ✅

**核心能力：**
- 基础四则运算：+ − × ÷
- 百分比计算 %
- 科学计算：sin/cos/tan（角度制）、平方根、log₁₀/ln、平方/立方、倒数、阶乘、绝对值、幂运算
- 退格、清空
- 科学计算模式可折叠/展开

**UI：**
- 大字体显示屏（40px）、科学计算折叠面板
- 按钮布局：标准计算器键盘布局 + 顶部科学计算行
- 运算符高亮（brand-50）、等号使用品牌色填充

**边界情况：**
- 除以 0 → 提示"除数不能为 0"
- 负数开平方 / 0 的倒数 → 提示
- 阶乘 > 170 → 提示"数值过大"

---

### 0.2 单位换算 ✅

**核心能力：**
- 8 大分类：长度、重量、温度、面积、体积、数据存储、速度、时间
- 每类 5-11 个单位，共计 60+ 单位
- 温度特殊处理（摄氏度 ↔ 华氏度 ↔ 开尔文 公式换算）
- 双向换算、一键切换源/目标单位

**支持单位列表：**

| 分类 | 单位 |
|---|---|
| 长度 | mm, cm, dm, m, km, in, ft, yd, mi, nmi, ly |
| 重量 | mg, g, kg, t, oz, lb, st |
| 温度 | °C, °F, K |
| 面积 | mm², cm², m², ha, km², in², ft², ac |
| 体积 | mL, L, m³, gal, qt, pt, cup, fl oz, in³, ft³ |
| 数据 | B, KB, MB, GB, TB, PB, KiB, MiB, GiB, TiB |
| 速度 | m/s, km/h, mph, kn, Mach |
| 时间 | ms, s, min, h, d, wk, mo, yr |

**边界情况：**
- 空输入 / 非法输入 → 结果为空
- 浮点数精度处理 → 保留 6 位有效数字，去除末尾 0

---

### 0.3 日期计算 ✅

**核心能力：**
- **日期差**：计算两个日期之间的天数差、工作日数（不含周六日）
- **日期加减**：任意加减年/月/日，正确处理闰年和月末（如 1月31日+1月→2月28/29日）
- **年龄计算**：从出生日期算周岁（年/月/日）

**输入：** DatePicker

**边界情况：**
- 月份加减超过 12 个月 → 自动进位年
- 目标月份天数不足 → 自动截断（如 3月31日+1月→4月30日）

---

### 0.4 倒计时 ✅

**核心能力：**
- 设定目标日期和时间
- 实时秒级刷新
- 自动判断未来/已过去
- 中文自然语言描述（"还剩 3 天 2 小时" / "已过去 1 年 2 个月"）

**UI：**
- 大数字显示（天以 72px 大字单独展示，时:分:秒 以 48px 展示）
- 已过时变为橙色（warning 色），未来时保持品牌色
- 底部显示目标日期

---

### 0.5 随机选择器 ✅

**核心能力：**
- **列表随机**：输入选项（逗号/换行分隔），随机选一个
- **抽签**：同上，可指定抽取数量（不重复）
- **随机数字**：指定范围，生成随机整数
- **抛硬币**：正面 / 反面
- **随机颜色**：生成随机 HEX 颜色，弹窗预览

**边界情况：**
- 仅有 1 个选项 → 直接返回该选项
- 空选项 → 提示

---

### 0.6 字数统计 ✅

参见原 3.1 节。从"文本处理"类移入日常工具。

## 一、格式化 & 转换类

### 1.1 JSON 工具 ✅

**核心能力：**
- 格式化（带缩进，可选 2/4 空格）
- 压缩（去空格换行，一行输出）
- 合法性校验（错误时显示错误信息）
- 示例按钮「填入示例 JSON」

**输入：**
- 文本输入框

**输出：**
- 格式化/压缩结果
- 校验失败时：红色错误提示

**边界情况：**
- 空输入 → 提示"请输入 JSON"
- 空对象/数组 → 正常工作

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 1.2 Base64 工具 ✅

**核心能力：**
- 文本 ↔ Base64 编解码
- URL Safe 模式（base64Url）

**编码输入：**
- 文本输入

**解码输入：**
- Base64 字符串

**边界情况：**
- 空输入 → 输出空字符串
- 非法 Base64 → 提示错误
- Unicode 字符（中文/Emoji）→ 正确处理 UTF-8

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 1.3 URL 编解码 ✅

**核心能力：**
- `encodeURIComponent` / `decodeURIComponent` — 编码 URL 参数
- `encodeURI` / `decodeURI` — 编码完整 URL（保留 `://` `/` `?` `#`）
- 实时输入输出（V2+）

**边界情况：**
- 空输入 → 输出空字符串
- 混合编码 → 部分解码 + 警告
- 中文字符 → 正确处理

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 1.4 时间戳工具 ✅

**核心能力：**
- 秒级/毫秒级时间戳 → 可读日期时间（自动识别精度）
- 多种格式输出：
  - ISO 8601
  - 中国常用: `2024-01-15 10:30:00`
  - 美国格式: `01/15/2024 10:30:00 AM`
  - 相对时间: `3 小时前` / `后天`
- 实时预览（输入任意一端，另一端即时更新）

**输入：**
- 时间戳输入框
- 日期时间文本框（自动解析多种格式）
- 快速填入「当前时间」

**边界情况：**
- 非法时间戳 → 忽略
- 空输入 → 展示当前时间

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 1.5 颜色工具 ✅

**核心能力：**
- 多格式互转：HEX ↔ RGB ↔ HSL ↔ HSV
- 模糊颜色名称匹配（常见 20 色）
- 解析任意格式：`#3B82F6` / `rgb(255,100,50)` / `hsl(200,80%,50%)`
- 颜色预览大色块 + 深色/浅色文字对比
- 快捷示例色块点击填入

**输入：**
- 任意格式颜色值输入
- 快捷示例 Chip

**输出：**
- 所有格式的转换结果
- 颜色名称
- 色块预览

**边界情况：**
- 简写 HEX (`#fff`) → 展开为 `#ffffff`
- 无 `#` 前缀 → 自动补全
- 非法颜色值 → 红色提示 + 无预览
- RGBA/HSLA 带透明度 → 支持 alpha 通道

**端侧：** ✅ 纯前端 · 🧩 可离线

## 二、编码 & 加密类

### 2.1 UUID / NanoID 生成器 ✅

**核心能力：**
- 批量生成（1-100 个）
- UUID v4（随机）/ v7（按时间排序）
- 格式选项：大写/小写/去横线/带花括号
- NanoID（已实现在工具库中，UI 层后续加入）

**输出：**
- 列表展示
- 一键复制全部 / 逐条复制

**边界情况：**
- 数量限制 1-100

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 2.2 哈希工具 ✅

**核心能力：**
- 6 种算法：MD5、SHA-1、SHA-224、SHA-256、SHA-384、SHA-512
- 实时计算（输入即更新）
- 大写/小写输出切换

**输入方式：**
- 文本输入

**输出：**
- 所有算法哈希值列表，每项可单独复制

**边界情况：**
- 空输入 → 输出各算法对空字符串的哈希（标准值）

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 2.3 JWT 解码器 ✅

**核心能力：**
- 解析 JWT token → 展示 Header / Payload / Signature
- Payload 美化展示（格式化 JSON）
- 自动检测过期时间 → 剩余有效时间 / 是否已过期
- 一键复制各段

**输入：**
- Token 粘贴框

**输出：**
- Header 解码
- Payload 解码 + 有效期状态条
- Signature

**边界情况：**
- 非法 token 格式（非三段式）→ 提示
- 过期 token → 红色警告 + 过期时间
- 未过期 → 绿色剩余时间

**端侧：** ✅ 纯前端 · 🧩 可离线

## 三、开发者工具类（原"文本处理"移入此处）

### 3.1 正则表达式测试器 ✅

**核心能力：**
- 输入正则 + 测试文本 → 实时匹配
- 匹配结果列表（序号 / 匹配内容 / 位置）
- Flags 开关：`g` `i` `m` `s` `u`
- 常用模式速查表（邮箱/URL/手机号/IP/日期/中文/空白行/Hex颜色）

**输入：**
- 正则输入框
- 测试文本多行输入
- Flags 开关按钮
- 常用模式 Chip 快捷填入

**输出：**
- 匹配结果列表
- 错误提示（非法正则时）

**边界情况：**
- 非法正则 → 实时错误提示
- 空正则 → 提示输入
- 大量匹配（>1000 个）→ 截断并提示

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 3.2 HTML 实体编解码 ✅

**核心能力：**
- 编码：`<` → `&lt;` 等 5 个预定义实体
- 解码：命名实体（HTML4）、十进制 `&#123;`、十六进制 `&#xABC;`

**边界情况：**
- 空输入 → 空输出
- 未知命名实体 → 保留原样

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 3.3 Unicode 码点查询 ✅

**核心能力：**
- 输入字符 → 显示 Unicode 码点（U+XXXX）
- 输入码点 → 显示对应字符（大字预览 48px）
- 显示基本信息：所属区块、名称
- 批量输入：一段文本 → 列出每个字符的码点

**边界情况：**
- Emoji 序列 → 列出每个码点
- 非法码点（U+110000+）→ 提示
- 控制字符（U+0000-U+001F）→ 显示名称

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 3.4 Cron 表达式解析 ✅

**核心能力：**
- 输入 Cron 表达式 → 解析为人类可读描述（中文）
- 预览接下来 5 次执行时间
- 支持标准 5 字段
- 常用预设 12 种：每分钟 / 每5分钟 / 每小时 / 每天0点 / 工作日9点等
- 支持缩写：`@hourly` `@daily` `@weekly` `@monthly` `@yearly`

**输入：**
- Cron 文本输入
- 快捷预设选择

**输出：**
- 人类可读描述（如"每天 0 点整"）
- 执行时间列表

**边界情况：**
- 非法表达式 → 错误提示
- `* * * * *` → "每分钟执行"

**端侧：** ✅ 纯前端 · 🧩 可离线

---

### 3.5 文字格式互转 ✅

**核心能力：**
- 大小写转换：大写 / 小写 / Title Case / Sentence case
- 命名法互转：camelCase / PascalCase / snake_case / kebab-case
- 反转字符串
- 实时转换

**输入/输出：** 文本框 + 各格式预览结果

**边界情况：**
- 空输入 → 所有输出为空
- 数字开头 → 保留
- 中文/混合文案 → 保留原样，只处理英文部分

**端侧：** ✅ 纯前端 · 🧩 可离线

## 七、UI/UX 规格（全局）

### 布局
- **Flutter (宽屏 ≥640px)**: `NavigationRail` + 工具列表（200px）+ 内容区 → 三栏
- **Flutter (窄屏 <640px)**: `NavigationBar`（底部 4Tab）+ `AppBar` + 工具内容 → 单栏
- **工具切换 (窄屏)**: AppBar 右侧列表按钮 → 底部抽屉

### 暗色模式
- 跟随系统 + 手动切换（AppTheme.light / AppTheme.dark）
- 所有工具支持深色/浅色

### 快捷键（V2+）
- `Cmd/Ctrl + K` → 全局搜索
- `Cmd/Ctrl + Enter` → 执行/格式化

### 响应式断点
| 断点 | 宽度 | 布局 |
|---|---|---|
| sm | < 640px | 手机：底部 Tab + 全宽内容 |
| md+ | ≥ 640px | 平板/桌面：Rail + 列表 + 内容 |
